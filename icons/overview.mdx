Skill Icons supports **200+ icons** spanning every major technology category — from programming languages and frontend frameworks to databases, DevOps tooling, editors, and cloud platforms. Whether you're building a portfolio README, a project landing page, or technical documentation, there's an icon for nearly every tool in your stack.

Each icon is referenced by a short, lowercase **icon ID** — for example, `js` for JavaScript, `react` for React, and `docker` for Docker. You pass one or more of these IDs to the `/icons` endpoint via the `?i=` query parameter to generate a dynamic SVG grid on the fly.

## Example: Mixed-Category Stack

Combine icon IDs from any category in a single request. The snippet below renders a badge for a full-stack TypeScript project:

```md
[![My Stack](https://skillicons.dev/icons?i=ts,react,nodejs,postgres,docker,aws)](https://skillicons.dev)
```

This produces a clickable SVG grid containing the TypeScript, React, Node.js, PostgreSQL, Docker, and AWS icons — ready to drop into any Markdown file, HTML page, or documentation site.
