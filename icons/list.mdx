The following table lists every icon ID currently supported by Skill Icons. Use these **exact lowercase IDs** in your `?i=` parameter when calling the [`/icons` endpoint](/api/get-icons). IDs are case-sensitive — always use lowercase.

<Tip>
  You can also fetch the live list programmatically via `GET https://skillicons.dev/api/icons`, which returns a JSON array of all current IDs. This is useful for build-time validation or auto-completing icon names in tooling.

  ```js
  fetch('https://skillicons.dev/api/icons')
    .then(res => res.json())
    .then(icons => console.log(`${icons.length} icons available:`, icons));
  ```
</Tip>
